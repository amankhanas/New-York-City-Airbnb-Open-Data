# New York City Airbnb Open Data Analysis

## Project Description

This project presents a complete data analytics workflow using the New York City Airbnb Open Data dataset. The project includes Exploratory Data Analysis (EDA), data cleaning, statistical analysis, data visualization, machine learning modeling, and an interactive Streamlit dashboard.

## Project Goal

The goal of this project is to identify factors affecting Airbnb rental prices in New York City and provide visual and analytical insights based on the dataset.

## Dataset

* Dataset Name: New York City Airbnb Open Data
* Source: Kaggle
* File: AB_NYC_2019.csv
* Rows: 48,895
* Columns: 16

### Main Features

* price
* neighbourhood_group
* neighbourhood
* room_type
* minimum_nights
* number_of_reviews
* reviews_per_month
* availability_365

## Research Questions

1. Which borough has the highest Airbnb prices?
2. How does room type affect price?
3. Which factors are most related to Airbnb prices?

## Results

The analysis shows that room type and neighbourhood significantly influence Airbnb prices. Manhattan contains the highest average rental prices, while entire homes are considerably more expensive than private rooms.
