# New York City Airbnb Open Data Analysis

## Must Have
- Очищенный датасет `data/processed/airbnb_clean.csv`
- Скрипты для очистки, анализа, визуализации и модели в `src/`
- Интерактивный дашборд в `app/streamlit_app.py`
- Документация в `README.md`

## Архитектура проекта
Проект построен по слоям, чтобы данные проходили через последовательный конвейер обработки, анализа и визуализации.

### Компоненты архитектуры
- **Data Layer**
  - `data/raw/` — исходный датасет `AB_NYC_2019.csv`
  - `data/processed/` — очищенные данные и результаты анализа
- **Processing Layer**
  - `src/data_cleaning.py` — загрузка, очистка и фильтрация данных, сохранение `data/processed/airbnb_clean.csv`
- **Analytics Layer**
  - `src/math_analysis.py` — описательная статистика и корреляционный анализ
  - `src/model.py` — подготовка признаков, обучение модели и оценка качества
- **Visualization Layer**
  - `src/visualization.py` — генерация графиков и сохранение изображений в `img/`
  - `app/streamlit_app.py` — интерактивный дашборд для визуального анализа
- **Documentation Layer**
  - `notebooks/` — исследовательские ноутбуки для EDA и экспериментов
  - `report/` — итоговый отчёт и выводы

### Поток данных
1. `data/raw/AB_NYC_2019.csv` — исходный датасет
2. `src/data_cleaning.py` — очищает данные и сохраняет `data/processed/airbnb_clean.csv`
3. `src/math_analysis.py` — анализирует очищенные данные и сохраняет результаты в `data/processed/*.csv`
4. `src/visualization.py` — генерирует графики и сохраняет PNG-файлы в `img/`
5. `app/streamlit_app.py` — загружает очищенный набор данных и показывает интерактивный дашборд

## Описание проекта
Аналитический проект на основе датасета New York City Airbnb Open Data. Проект включает очистку данных, EDA, математико-статистический анализ, визуализации, ML-модель и интерактивный Streamlit дашборд.

## Цель проекта
Выявить факторы, влияющие на цену аренды Airbnb в Нью-Йорке, и построить понятный аналитический продукт.

## Датасет
- Название: New York City Airbnb Open Data
- Источник: Kaggle
- Файл: `data/raw/AB_NYC_2019.csv`
- Строк: 48 895
- Признаки: 16

## Исследовательские вопросы
1. Какой район Нью-Йорка имеет наиболее высокие средние Airbnb-цены?
2. Как тип жилья влияет на стоимость аренды?
3. Какие признаки сильнее всего связаны с ценой аренды?

## Этапы выполнения
1. Загрузка данных
2. EDA
3. Очистка данных
4. Математико-статистический анализ
5. Визуализация данных
6. Разработка интерактивного дашборда
7. Построение ML-модели
8. Интерпретация результатов

## Структура проекта
- `data/raw/` — исходный датасет
- `data/processed/` — очищенный датасет
- `notebooks/` — ноутбуки EDA, анализа, ML
- `src/` — скрипты для очистки, анализа, модели и визуализации
- `app/` — Streamlit dashboard
- `img/` — сохраненные графики
- `report/` — финальный отчет

## Используемые технологии
Python, pandas, numpy, matplotlib, seaborn, plotly, scikit-learn, streamlit

## Как запустить проект
```bash
pip install -r requirements.txt
python src/data_cleaning.py
python src/visualization.py
streamlit run app/streamlit_app.py
```
