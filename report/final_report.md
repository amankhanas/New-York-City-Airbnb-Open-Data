# Финальный отчет: Анализ Airbnb в Нью-Йорке

## 1. Титульная информация
Проект: New York City Airbnb Open Data
Автор: Райқұл Арон Бердуалыұлы
Год: 2026

## 2. Цель проекта
Исследовать влияние характеристик объявлений Airbnb на цену аренды в Нью-Йорке и представить результаты в виде визуализаций, интерактивного дашборда и простой ML-модели.

## 3. Описание датасета
Датасет: New York City Airbnb Open Data (Kaggle)
Файл: AB_NYC_2019.csv
Строк: 48 895
Признаки: 16

## 4. Словарь признаков
- `price`: цена аренды, числовой
- `neighbourhood_group`: район, категориальный
- `room_type`: тип жилья, категориальный
- `minimum_nights`: минимальное число ночей, числовой
- `number_of_reviews`: количество отзывов, числовой
- `reviews_per_month`: отзывы в месяц, числовой
- `availability_365`: доступность в днях, числовой
- `latitude`, `longitude`: географические координаты

## 5. Первичный анализ данных / EDA
Проведен первичный анализ структуры данных, проверены типы признаков, выявлены пропуски и дубликаты. Были обнаружены пропуски в `name`, `host_name`, `last_review` и `reviews_per_month`, а также 0 дубликатов после очистки.

## 6. Очистка данных
- удалены дубликаты
- заполнены пропуски в `name` и `host_name`
- пропуски `reviews_per_month` заменены на 0
- приведен `last_review` к типу datetime
- удалены строки без координат
- отсечены экстремальные значения цены и выбросы по IQR
- очищенный датасет сохранен в `data/processed/airbnb_clean.csv`

## 7. Математико-статистический анализ
Проведен анализ основных числовых признаков и корреляционная матрица. Наибольшая положительная корреляция с ценой обнаружена у `availability_365` и `calculated_host_listings_count` в рамках выбранной очистки.

## 8. Основные визуализации
- график пропущенных значений
- гистограмма цен
- средняя цена по районам
- boxplot по типу жилья
- scatter plot цена vs количество отзывов
- тепловая карта корреляций

## 9. Интерактивный дашборд
Реализован Streamlit dashboard в `app/streamlit_app.py`.
Включает:
- KPI-карточки
- фильтры по району, типу жилья и цене
- интерактивную карту объектов
- график распределения цен
- среднюю цену по району
- scatter plot цены и отзывов
- boxplot по типу жилья
- корреляционную матрицу

## 10. ML-модель
Тип задачи: регрессия
Модель: Linear Regression
Целевая переменная: `price`
Использованные признаки: `minimum_nights`, `number_of_reviews`, `reviews_per_month`, `availability_365`, `calculated_host_listings_count`, `neighbourhood_group`, `room_type`

Метрики:
- MAE: средняя абсолютная ошибка
- MSE: средняя квадратичная ошибка
- RMSE: корень из MSE
- R2: доля объясненного разброса

## 11. Результаты и выводы
- Цена Airbnb значительно зависит от района и типа жилья.
- Manhattan показывает самые высокие средние цены.
- Entire home/apt дороже private room.
- Признаки отзывов и доступности помогают частично объяснить цену.
- Линейная регрессия дает базовый прогноз, но модель ограничена отсутствием расширенного feature engineering.

## 12. Ограничения
- Модель не учитывает сезонность и точные даты.
- Признаки `name` и `host_name` не использовались.
- Регрессия не всегда хорошо ловит масштабные выбросы по цене.

## 13. Запуск проекта
1. Установить зависимости:
```
pip install -r requirements.txt
```
2. Очистить данные:
```
python src/data_cleaning.py
```
3. Построить визуализации:
```
python src/visualization.py
```
4. Запустить дашборд:
```
streamlit run app/streamlit_app.py
```
