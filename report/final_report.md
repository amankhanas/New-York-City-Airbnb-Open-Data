# Финальный отчет: Анализ Airbnb в Нью-Йорке

## 1. Титульная информация
Проект: New York City Airbnb Open Data
Автор: Аманхан Алихан
Год: 2026

## 2. Цель проекта
Исследовать влияние характеристик объявлений Airbnb на цену аренды в Нью-Йорке и представить результаты в виде визуализаций, интерактивного дашборда и простой ML-модели.

## 3. Описание датасета
Датасет: New York City Airbnb Open Data (Kaggle)
Файл: AB_NYC_2019.csv
Строк: 48 895
Признаки: 16

## 4. Словарь признаков
- price: цена аренды, числовой признак
- neighbourhood_group: район, категориальный признак
- room_type: тип жилья, категориальный признак
- minimum_nights: минимальное число ночей, числовой признак
- number_of_reviews: количество отзывов, числовой признак
- reviews_per_month: отзывы в месяц, числовой признак
- availability_365: доступность в днях, числовой признак
- latitude, longitude: географические координаты

## 5. Первичный анализ данных / EDA
Проведён первичный анализ структуры данных, проверены типы признаков, выявлены пропуски и дубликаты. Основные проблемы датасета связаны с отсутствующими значениями в name, host_name, last_review и reviews_per_month. После очистки данные стали пригодны для анализа и обучения модели.

## 6. Очистка данных
- удалены дубликаты
- заполнены пропуски в name и host_name
- пропуски reviews_per_month заменены на 0
- last_review переведён в формат datetime
- удалены строки без координат
- отсечены экстремальные значения цены и выбросы по IQR
- очищенный датасет сохранён в data/processed/airbnb_clean.csv

## 7. Математико-статистический анализ
Выполнены описательные статистики для числовых признаков, рассчитана корреляционная матрица и оценена связь каждого признака с price. Для целевой переменной проведён анализ выбросов с помощью IQR и z-score. Наиболее информативными для дальнейшего анализа оказались признаки, связанные с ценой, отзывами и доступностью объекта.

## 8. Основные визуализации
- гистограмма распределения цен
- средняя цена по районам
- boxplot цен по типу жилья
- scatter plot цены и числа отзывов
- тепловая карта корреляций

## 9. Интерактивный дашборд
Реализован Streamlit-дашборд в app/streamlit_app.py. Он включает:
- KPI-карточки
- фильтры по району, типу жилья и цене
- интерактивную карту объектов
- график распределения цен
- среднюю цену по району
- scatter plot цены и отзывов
- boxplot по типу жилья
- корреляционную матрицу
- блок ML-анализа

## 10. ML-модель и визуальная интерпретация
Тип задачи: регрессия
Модель: Linear Regression
Целевая переменная: price
Использованные признаки: minimum_nights, number_of_reviews, reviews_per_month, availability_365, calculated_host_listings_count, neighbourhood_group, room_type

Метрики качества:
- MAE: средняя абсолютная ошибка прогноза
- MSE: средняя квадратичная ошибка
- RMSE: корень из MSE
- R²: доля объяснённого разброса целевой переменной

## 11. Аналитические сценарии
- сравнение средней цены по районам
- оценка влияния типа жилья на стоимость
- выявление признаков, наиболее связанных с ценой
- визуальная интерпретация ошибок модели

## 12. Основные выводы
- Цена аренды зависит от района, типа жилья и характеристик объявления.
- Манхэттен демонстрирует самые высокие средние цены.
- Entire home/apt обычно дороже, чем private room.
- Простая линейная регрессия даёт рабочий базовый прогноз, но имеет ограничения, связанные с линейной формой зависимости и отсутствием более сложного feature engineering.

## 13. Ограничения данных и модели
- модель не учитывает сезонность и временные закономерности
- не используются текстовые признаки из названия объявления
- линейная модель не всегда хорошо описывает сложные нелинейные зависимости

## 15. Запуск проекта
1. Установить зависимости:
```bash
pip install -r requirements.txt
```
2. Очистить данные:
```bash
python src/data_cleaning.py
```
3. Выполнить математический анализ:
```bash
python src/math_analysis.py
```
4. Обучить ML-модель и построить графики:
```bash
python src/model.py
python src/visualization.py
```
5. Запустить дашборд:
```bash
streamlit run app/streamlit_app.py
```
